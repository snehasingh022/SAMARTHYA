
```bash

```

### Build

```bash

```

## Tech Stack

- React 18
- Vite
- React Router
- TensorFlow.js (for computer vision)
- Framer Motion (for animations)
- Recharts (for progress visualization)

## Browser Requirements

- Modern browser with WebRTC support (for webcam features)
- HTTPS required for camera access (or localhost)

## Gemini API Integration (Optional)

For enhanced AI features:
- Intelligent speech evaluation in flashcards
- Enhanced object recognition with vision API
- Personalized learning recommendations
- Progress analysis and insights

See [GEMINI_SETUP.md](./GEMINI_SETUP.md) for setup instructions.

**Note:** The application works fully without the Gemini API key - all features have fallback modes. Gemini enhances the experience but is not required.
